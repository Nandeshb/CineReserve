# 🎬 CineReserve - Movie Ticket Booking System

## 📌 Overview

CineReserve is a full-stack Movie Ticket Booking System developed using *Python, Flask, and MySQL*. It enables users to browse movies, book tickets, select seats, and manage bookings, while administrators can manage movies, showtimes, and users.

---

## ✨ Features

### User Features
User Registration & Login
Browse Movies
View Show Timings
Seat Selection
Book Movie Tickets
Booking History
Profile Management

### Admin Features
Admin Login
Add/Edit/Delete Movies
Manage Show Timings
Manage Users
View Bookings
Role-Based Access Control

## 🚀 Installation

### Clone the Repository

bash
git clone https://github.com/Nandeshb/CineReserve.git


### Navigate to the Project

bash
cd CineReserve


### Install Dependencies

bash
pip install -r requirements.txt


### Configure MySQL

Create a MySQL database.
Update your database credentials in app.py.

### Run the Application

bash
python app.py


Open your browser and visit:


http://127.0.0.1:5000
